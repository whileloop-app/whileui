## TL;DR

- **Blocks:** Auth, Navigation, Layout, Profile, Lists, Commerce, Splash, Media
- **Auth blocks have callbacks:** `onSubmit`, `onForgotPassword`, `onSignUp`, etc. Wire them.
- **Full-screen:** Use `AppShell` + `Header` + `BottomNav` + content.
- **Layout:** Use `Stack` (vertical) and `Row` (horizontal) with `gap`, `align`, `justify`.

See [docs/AI_GUIDE.md](./docs/AI_GUIDE.md) for details.
